# 📌 Project Overview

This repository contains **10 practical implementations** based on the subject:

📘 **Data Analytics Using Tools (DAT)**  

The project focuses on applying:

- Data Analytics fundamentals  
- Data preprocessing techniques  
- Data integration from multiple sources  
- Exploratory Data Analysis (EDA)  
- Predictive modeling  

---

# 🎯 Objective

- Understand real-world data analytics workflow  
- Apply data cleaning and transformation techniques  
- Perform analysis using structured datasets  
- Build predictive models  

---

# 🧠 Learning Outcomes

- Data classification and categorization  
- Data sources and repositories  
- Data cleaning and preprocessing  
- Data integration  
- Statistical analysis  
- Predictive modeling (Regression)

# ⚙ Installation

Clone repository

```
git clone https://github.com/AlokRana01/data-analytics-practicals-dat.git
```

Go to project folder

```
cd data-analytics-practicals-dat
```

Install dependencies

```
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

---

# ▶ How to Run

Open Jupyter Notebook

```
jupyter notebook
```

Then open any practical:

```
practical_1.ipynb
```
